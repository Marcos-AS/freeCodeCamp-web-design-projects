Introducción

Python es un lenguaje de programación potente y fácil de aprender. Tiene estructuras de datos de alto nivel eficientes y un simple pero efectivo sistema de programación orientado a objetos. La elegante sintaxis de Python y su tipado dinámico, junto a su naturaleza interpretada lo convierten en un lenguaje ideal para scripting y desarrollo rápido de aplicaciones en muchas áreas, para la mayoría de plataformas.

El intérprete de Python y la extensiva librería estándar se encuentran disponibles libremente en código fuente y forma binaria para la mayoría de plataformas desde la Web de Python, www.python.org, y se pueden distribuir libremente. El mismo sitio contiene distribuciones y direcciones a muchos módulos de Python de terceras partes, programas, herramientas y adicionalmente documentación.

El intérprete de Python es fácilmente extensible con funciones y tipos de datos implementados en C o C++ (o otros lenguajes que permitan ser llamados desde C). Python también es apropiado como un lenguaje para extender aplicaciones modificables.

Los datos de Python son objetos

Python sigue lo que llamamos modelo de objetos, ¿qué significa eso?
Cada número, cadena, estructura de datos, función, clase, módulo, etc. existe en el intérprete de Python en su propia "caja", que se conoce como un objeto de Python. Cada objeto tiene una identidad, un tipo y un valor. Por ejemplo, cuando escribe a = 42, se crea un objeto entero con el valor 42.
En Python, un objeto es un fragmento de datos que contiene al menos lo siguiente:

  • Un tipo que define lo que puede hacer
  • Una identificación única para distinguirlo de otros objetos
  • Un valor acorde con su tipo
  • Un recuento de referencia que rastrea la frecuencia con la que se usa este objeto

Comentarios

Para escribir un comentario en Python y así poder entender nuestro propio código y permitir que los demás también lo entiendan, se puede escribir un # antes del comentario si este es de una sola línea, o """ si el comentario es de más líneas de extensión:

#este es un comentario corto, de una sola línea
"""este es
un comentario
largo"""
Variables

Las variables son creadas cuando se les asigna un valor por primera vez, en realidad Python le da la referencia del valor a la variable. Se pueden asignar valores o el mismo valor a múltiples variables en una línea.

#defino una variable
variable0 = 10
#asigno valores a múltiples variables en una sola definición
variable1, variable2 = "bananas", "manzanas"
#asigno el mismo valor a múltiples variables
variable3, variable4 = True

Cuando se le asigna un valor a una variable, automáticamente el lenguaje detecta el tipo de dato y declara la variable como tal, por lo que no se necesita declarar el tipo de variable e incluso puede cambiar el tipo después de haber sido creada.

Para conocer el tipo de variable, se puede usar la función built-in type()

if type(variable0) is int:
print('Es un entero.')

Salida:

Es un entero.
Operadores

Aparte de las funciones incorporadas, los operadores usados para hacer operaciones sobre variables y valores son:

  • Aritméticos: usados con valores numéricos para hacer operaciones matemáticas comunes, también para concatenar texto.
  • De asignación: para asignar valores a variables.
  • De comparación: para comparar valores.
  • Lógicos: para combinar oraciones condicionales.
  • De identidad: para comparar objetos, no si son iguales, sino si son de hecho el mismo objeto, con el mismo lugar en la memoria.
  • De pertenencia: para comprobar si una secuencia está presente en un objeto.
  • Bit a bit: para comparar números binarios.
